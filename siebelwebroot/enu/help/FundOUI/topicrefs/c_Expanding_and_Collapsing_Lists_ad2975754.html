Expanding and Collapsing Lists

Every list has a show more (expand) button or a show less (collapse) button in the corner. Click Show More or Show Less to see more or fewer records in a list. The following table describes the show more and show less buttons.

This table shows each (expand/collapse) button, the corresponding button name and description.

Button

Name

Description

Show More

Displays more records to expand a collapsed list.

Show Less

Displays fewer records to collapse an expanded list.
