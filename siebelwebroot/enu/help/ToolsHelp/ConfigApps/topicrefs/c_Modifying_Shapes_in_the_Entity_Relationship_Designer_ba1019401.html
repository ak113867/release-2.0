Modifying Shapes in the Entity Relationship Designer

This topic describes how to modify shapes in the Entity Relationship Designer. You perform all tasks that this topic describes in the Entity Relationship Designer.

To modify shapes in the Entity Relationship Designer

  1. To modify the appearance of a shape in an ERD:

    1. Choose an entity or relationship.

    2. Right-click, and then click Shape Properties.

    3. Modify the properties in the Item Properties dialog box, and then click OK.

  2. To choose multiple objects in an ERD:

    1. Choose an entity.

    2. Hold down the shift key.

    3. With the shift key still depressed, click another entity.

    4. Release the shift key.

  3. To align shapes relative to each other:

    1. Choose multiple objects in the Entity Relationship Designer.

      For more information, see the following.

    2. Right-click the canvas, click Layout, Align, and then click one of the following menu items:

      • Lefts

      • Centers

      • Rights

  4. To make shapes the same size:

    1. Choose multiple objects in the Entity Relationship Designer.

      For more information, see Modifying Shapes in the Entity Relationship Designer.

    2. Right-click the canvas, click Layout, Make Same Size, and then click one of the following menu items:

      • Width

      • Height

      • Both
