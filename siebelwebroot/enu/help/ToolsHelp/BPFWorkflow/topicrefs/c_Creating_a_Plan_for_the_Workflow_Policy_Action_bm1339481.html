Creating a Plan for the Workflow Policy Action

This task is a step in Process of Planning a Workflow Policy.

If the Workflow Policy conditions are met, then Siebel CRM runs a Workflow Policy action. There are a number of possible Workflow Policy actions provided out-of-the-box. You can use these actions or define your own custom actions.
