Workflow Process Program

The definition of an event. Types of events include Send Email, Send Page, Database Operation, Send Broadcast Message, and Run External Program. Different properties are associated with a Workflow Process program depending on the event type. Some of the properties that you can define for a program include the fields that Siebel CRM can substitute into a message, the possible recipients of a message, and the Siebel database columns that it must update.
